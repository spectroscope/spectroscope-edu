the edu portal · handbookdas edu-portal · handbuch

everything the simulator and the lessons can doalles, was der simulator und die lektionen können

spectroscope is the agent orchestrator you can watch. this app teaches what you are watching, two ways: a simulator that replays a scripted agent run on a live system map, and a set of lessons that build an agent up one concept at a time. no backend, no key, nothing to install. this handbook covers both.

spectroscope ist der agent-orchestrator, den man beobachten kann. diese app lehrt, was du beobachtest, auf zwei arten: ein simulator, der einen gescripteten agenten-lauf auf einer lebenden system-map abspielt, und eine reihe lektionen, die einen agenten konzept für konzept aufbauen. kein backend, kein key, nichts zu installieren. dieses handbuch deckt beides ab.

01reading the mapdie karte lesen

both the simulator and the lessons draw the same system map. learn it once and you read it everywhere.

der simulator und die lektionen zeichnen dieselbe system-map. lern sie einmal, und du liest sie überall.

  • your mac vs. outsidedein mac vs. außerhalb: a boundary splits what runs on your machine from what lives beyond the network. the llm sits inside when it is local, outside when it is remote.: eine grenze trennt, was auf deiner maschine läuft, von dem, was jenseits des netzes lebt. das llm sitzt innen wenn lokal, außen wenn remote.
  • agent: the hub. it shows the loop (plan, call tools, read results), the permission gate, the tool belt, and the live context sent to the llm.: die zentrale. sie zeigt die loop (planen, tools aufrufen, ergebnisse lesen), das permission-gate, den tool-gürtel und den live-kontext, der ans llm geht.
  • operating system bandbetriebssystem-band: the stations the agent reaches through, disk, shell, mcp-client and the network stack.: die stationen, durch die der agent greift: disk, shell, mcp-client und der netzwerk-stack.
  • llm: the model. it only reads and writes text; everything around it is the harness.: das modell. es liest und schreibt nur text; alles darum herum ist der harness.
  • subagents / workerssubagenten / worker: forked agents, each with its own context window, for parallel or isolated work.: abgespaltene agenten, jeder mit eigenem kontextfenster, für parallele oder isolierte arbeit.

the railsdie rails

  • active railaktives rail: where it is happening right now; a packet rides it.: wo es gerade passiert; ein paket fährt darauf.
  • inside the agent systemim agenten-system: a link that stays on your mac.: eine verbindung, die auf deinem mac bleibt.
  • outbound: dashed; a leg that crosses the network boundary.: gestrichelt; ein bein, das die netzgrenze überquert.

every event also carries a color, the same one you meet in the trace: token, tool, gate, subagent, reasoning.

jedes ereignis trägt auch eine farbe, dieselbe, die dir im trace begegnet: token, tool, gate, subagent, reasoning.

02the simulatorder simulator

a scripted agent run you steer. it replays real events with no backend, so you can stop, scrub and read every step.

ein gescripteter agenten-lauf, den du steuerst. er spielt echte ereignisse ab, ohne backend, du kannst also stoppen, scrubben und jeden schritt lesen.

the transportder transport

  • the now banddas now-band: over the map, it always names the current station (the model is thinking, at the permission gate, done).: über der karte nennt es immer die aktuelle station (das modell denkt, am permission-gate, fertig).
  • the replay bardie replay-leiste: drag it to scroll the run to any step. reset, previous, play and next sit beside it.: zieh sie, um den lauf zu jedem schritt zu scrollen. reset, zurück, play und vor liegen daneben.
  • jsonl + trace drawersjsonl- + trace-drawer: open the left and right panels to read the same run as the raw event stream and as the causal trace. both move in lockstep with the map.: öffne das linke und rechte panel, um denselben lauf als rohen event-strom und als kausalen trace zu lesen. beide laufen im gleichschritt mit der karte.
  • local / remote fliplokal / remote-schalter: switch the model between a local one on your mac and a hosted one beyond the boundary, and watch the llm card cross. remote is the default.: schalte das modell zwischen einem lokalen auf deinem mac und einem gehosteten jenseits der grenze um und sieh die llm-karte hinüberwandern. remote ist der standard.
  • drag a cardkarte ziehen: move any card and it stays put across every step; only cards you did not touch re-arrange.: verschieb eine karte und sie bleibt über alle schritte liegen; nur karten, die du nicht angefasst hast, ordnen sich neu.

the scenariosdie szenarien

the sidebar lists scripted runs to steer: a build_plan delegation, a three-lens code review, a permission-gate walk (blocked and allowed), disk and shell, a coding fan-out, a research consolidation, a context-window fill, a dark-mode plan. pick one and step it.

die seitenleiste listet gescriptete läufe zum steuern: eine build_plan-delegation, ein drei-linsen-code-review, ein permission-gate-durchlauf (blockiert und erlaubt), disk und shell, ein coding-fan-out, eine research-konsolidierung, ein kontextfenster-füllen, ein dark-mode-plan. wähl einen und steppe ihn.

03the lessonsdie lektionen

short, interactive lessons that build an agent up one concept at a time, on the same cards you meet in the simulator.

kurze, interaktive lektionen, die einen agenten konzept für konzept aufbauen, auf denselben karten, die dir im simulator begegnen.

  • step or scrubsteppen oder scrubben: use the arrows, play, or the scrub bar. each step advances one teaching beat, with a caption that explains what just happened.: nutz die pfeile, play oder die scrub-leiste. jeder schritt bringt einen lehr-takt weiter, mit einer bildunterschrift, die erklärt, was gerade passiert ist.
  • predict then revealvorhersagen, dann aufdecken: some steps ask you to guess first (which verdict? which skill loads?) and reveal the answer after you choose.: manche schritte lassen dich erst raten (welches urteil? welcher skill lädt?) und decken die antwort nach deiner wahl auf.
  • the readout raildie readout-leiste: on the right, it tracks the lesson's meter: a context ring, a token budget, the growing session.jsonl, or what the harness gives you.: rechts verfolgt sie den messwert der lektion: einen kontext-ring, ein token-budget, das wachsende session.jsonl oder was der harness dir gibt.

the fifteen lessonsdie fünfzehn lektionen

anatomy of an agent

a raw model becomes a working agent, one harness layer at a time.aus einem rohen modell wird ein arbeitender agent, schicht für schicht.

inside the context window

the window fills to rot, then the harness compacts it.das fenster füllt sich bis zur fäulnis, dann kompaktiert der harness.

one turn through the machine

assemble, decide, gate, execute, observe, repeat.zusammenbauen, entscheiden, gate, ausführen, beobachten, wiederholen.

the gate decides

allow, ask or deny, read off the blast radius. predict each.erlauben, fragen oder ablehnen, am blast-radius. rate jedes.

the four moves on the window

select, compress, isolate, write, and the budget never grows.select, compress, isolate, write, und das budget wächst nie.

the agent draws

generate_image, a multimodal action on the same loop and gate.generate_image, eine multimodale aktion auf derselben loop und gate.

thin until it matches

progressive disclosure, load context only when it is needed.progressive disclosure, kontext nur bei bedarf laden.

a skill is knowledge on demand

a folder loaded on match, into context, no weights change.ein ordner, bei match geladen, in den kontext, ohne gewichts-änderung.

mcp: tools from another process

the call rides client, network, server, and the gate still applies.der aufruf fährt client, netz, server, und das gate gilt weiter.

serial vs. orchestrated

fan out to isolated workers, each with its own window.fächere zu isolierten workern auf, jeder mit eigenem fenster.

a fleet that reaches out

workers that each call an mcp server from their own window.worker, die je einen mcp-server aus dem eigenen fenster rufen.

a real feature, many turns

agency is many small verified steps sustained over a task.agency ist viele kleine, verifizierte schritte über eine aufgabe.

a hook is a guarantee

a checkpoint that fires every time, blocking or injecting.ein checkpoint, der jedes mal feuert, blockt oder einspeist.

the extensions compose

hook, mcp and skill layered together in one run.hook, mcp und skill in einem lauf zusammengelegt.

the spec survives the session

durable decisions written to disk outlive the window.dauerhafte entscheidungen auf disk überleben das fenster.

05concepts, in one line eachkonzepte, je in einer zeile

the vocabulary the lessons teach, gathered here for reference.

das vokabular, das die lektionen lehren, hier zum nachschlagen gesammelt.

  • the loopdie loop: the model emits one thing; the harness runs it, feeds reality back, and repeats until the turn ends.: das modell gibt eine sache aus; der harness führt sie aus, speist die realität zurück und wiederholt, bis die runde endet.
  • the context windowdas kontextfenster: a fixed budget of tokens, rebuilt every turn; when it fills, quality rots, so the harness compacts it.: ein festes token-budget, jede runde neu gebaut; wenn es voll wird, verrottet die qualität, also kompaktiert der harness.
  • the four movesdie vier operationen: select in, compress down, isolate away, write out, none of which enlarges the window.: select rein, compress runter, isolate weg, write raus, keine vergrößert das fenster.
  • the permission gatedas permission-gate: the one write path; every side-effecting call is allowed, asked, or denied, and every verdict is logged.: der eine schreibpfad; jeder aufruf mit nebenwirkung wird erlaubt, erfragt oder abgelehnt, und jedes urteil wird geloggt.
  • tools: read_file, write_file, run_command and more, a registry the harness validates, executes, and feeds back.: read_file, write_file, run_command und mehr, ein register, das der harness prüft, ausführt und zurückspeist.
  • mcp: the model context protocol, tools that live in a separate server process and plug in over a standard protocol.: das model context protocol, tools, die in einem separaten server-prozess leben und über ein standard-protokoll andocken.
  • skill: a folder (SKILL.md plus resources) the harness loads only when the task matches, adding expertise without changing the model.: ein ordner (SKILL.md plus ressourcen), den der harness nur bei passender aufgabe lädt, expertise ohne änderung am modell.
  • hook: a guaranteed checkpoint that fires on an event and can block a call or inject context.: ein garantierter checkpoint, der auf ein ereignis feuert und einen aufruf blocken oder kontext einspeisen kann.
  • subagents / fleetsubagenten / flotte: forked agents with their own windows; a fleet fans work out and merges only the results back.: abgespaltene agenten mit eigenen fenstern; eine flotte fächert arbeit auf und führt nur die ergebnisse zurück.
  • session.jsonl: the append-only log of the whole run, replayable line by line, the reason any run can be watched.: das append-only-log des ganzen laufs, zeile für zeile abspielbar, der grund, warum jeder lauf beobachtbar ist.

06good to knowgut zu wissen

  • backend-freebackend-frei: every run is a scripted replay; there is no server, no key, and no model call. it is deterministic and scrubbable.: jeder lauf ist ein gescriptetes replay; kein server, kein key, kein modell-aufruf. deterministisch und scrubbar.
  • three themes, two languagesdrei themes, zwei sprachen: switch dark, paper or white and english or german from the sidebar; this handbook follows the same toggles above.: schalte dark, paper oder white und englisch oder deutsch aus der seitenleiste; dieses handbuch folgt denselben schaltern oben.
  • runs from filesläuft aus dateien: the whole app is static assets, local-first, the way the product itself runs.: die ganze app sind statische dateien, local-first, so wie das produkt selbst läuft.
